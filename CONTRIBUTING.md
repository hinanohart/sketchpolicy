# Contributing to sketchpolicy

Thanks for your interest. sketchpolicy is a pre-alpha bootstrap scaffold; the bar
for merging is "honest, tested, and CPU-only by default".

## Ground rules

- **Core stays torch-free.** The default install must resolve to
  numpy / scipy / pyarrow only. Anything heavier (mediapipe, pybullet,
  lerobot-dataset) goes behind an optional extra.
- **No silent network I/O in library code.** Models and data are provisioned by
  the user; the library only reads local paths.
- **Honest marketing is enforced in code.** The single source of truth for the
  project's claims, disclaimer and the list of banned overclaim phrases is
  `src/sketchpolicy/contract.py`. The tests in `tests/test_contract.py` and
  `tests/test_readme.py` scan the README and disclaimer with an exact-string
  check and a negative fixture, so a new overclaim ("state of the art",
  "production ready", a policy-success promise, etc.) will fail CI. Do not paste
  example marketing copy containing those phrases into scanned files — describe
  the policy here instead and add the phrase to the list in `contract.py`.
- **Numbers are measured.** Any figure in the README must come from
  `bench_results/*.json`, regenerated by `python scripts/measure.py`. CI does not
  regenerate the bench file, so update both together.

## Development

```bash
python -m venv .venv && . .venv/bin/activate
pip install -e ".[dev,replay]"
pytest -q
python scripts/verify_step.py S1   # per-phase artifact + test check
```

The optional `[ingest]` path additionally needs MediaPipe plus system GL
libraries (e.g. `libgles2`, `libegl1` on Debian/Ubuntu) and the
`hand_landmarker.task` model; `sketchpolicy doctor` reports what is available.

## Scope

v0.1 augments the proprioceptive / action stream (EE pose + gripper) only, with
no IK, no video transform, parallel-jaw grippers, and a planar position default.
Please keep PRs within that honest scope, or open an issue to discuss extending
it (e.g. multi-file chunking, optical-flow video augmentation, dexterous hands).
